### Prerequisites

- Node.js 18+ 
- npm atau yarn

### Setup

```bash
# Clone repository
git clone https://github.com/kelasphp-kafekoding/website.git
cd website

# Install dependencies
npm install

# Jalankan development server
npm run dev

# Build untuk production
npm run build
```
