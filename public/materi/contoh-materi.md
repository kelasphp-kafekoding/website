# Pengenalan PHP

## Apa itu PHP?

PHP (Hypertext Preprocessor) adalah bahasa pemrograman server-side yang populer untuk pengembangan web. PHP dapat digunakan untuk membuat website dinamis dan interaktif.

## Keunggulan PHP

- **Mudah dipelajari**: Sintaks yang sederhana dan mirip dengan bahasa pemrograman lain
- **Open Source**: Gratis dan memiliki komunitas yang besar
- **Cross-platform**: Dapat berjalan di berbagai sistem operasi
- **Database Support**: Mendukung berbagai database seperti MySQL, PostgreSQL, dll

## Contoh Kode PHP

```php
<?php
echo "Hello, World!";

// Variabel
$nama = "Kafekoding";
$umur = 5;

// Array
$teknologi = ["PHP", "MySQL", "JavaScript"];

// Function
function sapa($nama) {
    return "Halo, " . $nama . "!";
}

echo sapa("Peserta");
?>
```

## Instalasi PHP

1. Download PHP dari [php.net](https://www.php.net)
2. Install sesuai sistem operasi
3. Verifikasi instalasi dengan `php -v`

## Kesimpulan

PHP adalah pilihan yang tepat untuk memulai belajar web development karena mudah dipelajari dan memiliki banyak resource pembelajaran.
